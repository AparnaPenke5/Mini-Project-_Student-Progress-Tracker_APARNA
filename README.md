# Mini-Project-_Student-Progress-Tracker_APARNA
“A Python console-based Student Progress Tracker that records student details, calculates total and average marks, assigns grades, determines pass/fail status, and provides a menu-driven interface.”  If you want it even shorter or more beginner-focused, tell me and I’ll tighten it further.
Features

Input student details (Name, Class, Roll Number)

Enter marks for any number of subjects

Automatic calculation of:

Total marks

Average marks

Grade assignment based on average score

Automatic failure if any subject score is below 35

Clear and formatted student progress report

Menu options to:

Enter marks again

View the report again

Exit the program

Programming Concepts Used

while and for loops

Conditional statements (if, elif, else)

Boolean flags for validation logic

User input handling

Formatted output using f-strings

Menu-driven program structure

How to Run

Ensure Python 3.x is installed on your system

Clone or download this repository

Navigate to the project directory

Run the program using:

python student_progress_tracker.py
Grade Evaluation Criteria
Average Score	Grade
90 and above	A
75 – 89	B
50 – 74	C
35 – 49	D
Below 35	Fail
